🔗 Google Sheets Sync Setup

Connect this app to your Google Sheet.
One-time setup · takes about 3 minutes.
After setup, tap "Sync to Google Sheets" on the Add screen to push all entries directly into your 📒 Expenses sheet.

  1. Open your Google Sheet in Chrome → tap ⋮ → Share & export → Script editor
  2. Delete everything in the editor. Tap the code box below to copy it, then paste it in the editor
  3. Tap the 💾 Save icon (or Ctrl+S)
  4. Tap ▶ Run → tap Review permissions → Allow when asked
  5. Tap Deploy (top right) → New deployment
    Type: Web app · Execute as: Me · Who has access: Anyone
    Tap Deploy
  6. Copy the Web app URL → paste it in the field below → tap Save
